پروژه ی آزمون نرم افزار

تولید داده های آزمون به پنج روش مقدار مرزی، استحکام، بدترین حالت، بدترین حالت قوی و کلاس هم ارزی

برنامه ی هدف: یک بازی با تاس می باشد

تعداد خط های برنامه ی هدف: ۱۰۰۰ خط

ورودی های برنامه هدف: سه عدد صحیح به عنوان ورودی می گیرد که عدد اول، تعداد سطر و ستون می باشد(بازه ی معتبر بین ۲ تا ۲۰)؛ عدد دوم، تعداد بازیکن ها می باشد(بازه ی معتبر بین ۲ تا ۴)؛ عدد سوم، تعداد بمب های بازی(بازه ی معتبر از ۰ تا نصف خانه های ماتریس) می باشد.

توضیح پروژه:

برنامه نوشته شده داده های آزمون به پنج روش گفته شده تولید می کند و به صورت خودکار داده های آزمون تولید شده را روی برنامه ی هدف آزمایش می کند و خروجی برنامه ی هدف را در هر مرحله در فایل ذخیره می کند. نتیجه کلی آزمایش در هر پنج روش در فایلی ارایه می شود.

موارد مورد نیاز برای اجرای پروژه

  1. نصب بودن زبان برنامه نویسی پایتون ۳ روی سیستم خود درصورت نصب نبودن فایل پایتون را از آدرس https://www.python.org/downloads/ دریافت کنید و نصب کنید.
  2. نصب بسته های پایتون استفاده شده در پروژه به روش زیر
    • python -m pip install -U pip setuptools
    • pip install random
    • pip install itertools
    • pip install csv
    • pip install webbrowser
    • pip install subprocess

روش اجرای برنامه

به مسیر پروژه بروید و فایل Main.py را اجرا کنید:

> python Main.py

خروجی های برنامه

  1. تست کیس های تولید شده در هر آزمون در فایل های csv ذخیره شده است.
    • output_BVA.csv تست کیس های آزمون مقدار مرزی
    • output_STH.csv تست کیس های آزمون استحکام
    • output_WST.csv تست کیس های آزمون بدترین حالت
    • output_WST_STG.csv تست کیس های آزمون بدترین حالت قوی
    • output_EC.csv تست کیس های آزمون کلاس هم ارزی
  2. در فایل final_result.txt نتیجه کلی تمام آزمون ها ارایه شده است.
  3. در مسیر outputs ، فایل های موجود در این پوشه خروجی های برنامه ی هدف در هر داده آزمون می باشد.
  4. نام فایل های موجود در پوشه ی outputs به این شکل می باشد:
    • outputs_نام آزمون_داده ی اول_داده ی دوم_داده ی سوم.txt
